3. Description du service

DataXtrak est un logiciel de bureau permettant d'importer des documents, d'en extraire automatiquement des informations à l'aide de traitements d'intelligence artificielle, et d'organiser ces informations dans des tableurs structurés. Le Logiciel s'exécute localement sur l'ordinateur de l'utilisateur.

Logiciel en cours de développement : le Logiciel est fourni dans une version évolutive. Certaines fonctionnalités peuvent être ajoutées, modifiées ou retirées. Il est possible que des dysfonctionnements subsistent.

4. Fonctionnalité de création de brouillons d'e-mails

Le Logiciel propose, de façon optionnelle, de créer des brouillons d'e-mails dans la messagerie de l'utilisateur (Gmail, Microsoft Outlook) à partir des données extraites. Cette fonctionnalité nécessite que l'utilisateur autorise l'accès via le mécanisme officiel du fournisseur de messagerie.

  • L'accès est limité à la création de brouillons. Le Logiciel ne lit pas et n'envoie pas les e-mails de l'utilisateur.
  • L'utilisateur peut révoquer cette autorisation à tout moment depuis son compte Google ou Microsoft.
  • L'utilisation des données issues des API Google respecte la Google API Services User Data Policy (usage limité).
